### File Purposes

#### jest.setup.js
- Executes as Jest lifecycle hook
- Creates: assets/js/life-money/ directory
- Creates: test/unit/assets/js/life-money/ directory
- Writes: compute.js, chart.js, main.js source files
- Logs: "Jest setup: all life-money modules created"

#### run-tests.js
- Standalone Node.js script that doesn't require PowerShell
- Creates directories and source files (same as jest.setup.js)
- Executes: npx jest --watchAll=false
- Can be run directly: node run-tests.js

#### run-tests.bat
- Windows batch file wrapper for run-tests.js
- Executes: node run-tests.js from the project root

#### package.json Changes
- Added: "setupFilesAfterEnv": ["<rootDir>/jest.setup.js"]
- This tells Jest to load jest.setup.js before running any tests

### Import Path Resolution

Test files import from relative paths:
```javascript

// Resolves to: assets/js/life-money/compute.js

// chart.test.js (at: test/unit/assets/js/)
```
